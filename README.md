# HỆ THỐNG ĐẶT VÉ XEM PHIM CGV

## Mô tả
Website hỗ trợ đặt vé xem phim trực tuyến, cho phép:
- Khách vãng lai xem phim, tìm kiếm, xem khuyến mãi
- Người dùng đăng ký, đăng nhập, đặt vé và thanh toán
- Quản trị viên quản lý phim, người dùng, khuyến mãi, doanh thu

## Môn học
Phân tích thiết kế hệ thống hướng đối tượng
